## Tinkercad Traffic Light Simulator
An Arduino-based traffic light simulator built and tested in Autodesk Tinkercad. This project replicates a standard four-way or single-intersection traffic signal sequence using LEDs and timed delays.
## Features

* Standard Sequence: Red $Green $Yellow $Red loops.
* Tinkercad Ready: Designed for easy importing and simulation.
* Precise Timing: Uses optimized delay() functions for realistic intervals.
* Modular Code: Clean, well-commented Arduino C++ script.

## Hardware Components

* 1x Arduino Uno R3
* 1x Red LED (Stop)
* 1x Yellow LED (Caution)
* 1x Green LED (Go)
* 3x $220 $ Resistors
* 1x Breadboard
* Jumper Wires

## Setup Instructions
## 1. Tinkercad Simulation

   1. Log in to Tinkercad Circuits.
   2. Create a new Circuit design.
   3. Wire the components according to the pin configuration below.
   4. Open the Code panel, switch to Text mode, and paste the code from traffic_light.ino.
   5. Click Start Simulation.

## 2. Pin Configuration

* Red LED: Pin 13
* Yellow LED: Pin 12
* Green LED: Pin 11
* GND: Common ground via resistors
